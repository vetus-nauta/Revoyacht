# Локальные отчеты

Локальная программа для ведения финансовых отчетов на этом компьютере.

## Запуск

```bash
./start-report-client.sh
```

Программа откроет отдельное окно Chrome. Данные хранятся в `reports.sqlite3`, Excel-файлы создаются в папке `exports`.

## Что умеет

- дата создания отчета;
- остаток на дату;
- приходы;
- расходы;
- предстоящие расходы;
- выводы: пришло, ушло, остаток, будущий остаток с учетом расходов;
- создание `.xlsx`;
- отправка `.xlsx` по email через SMTP, если указаны настройки почты.

Пароль SMTP не сохраняется.
