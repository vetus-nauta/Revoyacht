## Scope

- frontend implementation efficiency;
- input responsiveness;
- mobile Safari keyboard behavior;
- iPad keyboard/viewport behavior;
- minimal dependency discipline;
- interaction performance;
- feed performance;
- attachment interaction performance;
- PWA readiness later.

- create dashboard-first UX;
