# Sprint 02 — Clean Core Foundation

## Goal

Create the new clean technical foundation for FinDesk v2.0.

## Depends on

Sprint 01 final report.

## Director rule

Use a new Director. Start from project docs and Sprint 01 handoff only.

## Agents

- Data and Backend Core Agent
- Financial Logic Agent as reviewer
- QA/Audit Agent

## Scope

- apply clean schema;
- create workspace model;
- create member model;
- create Cash/Card flows;
- create entries model;
- load category seed;
- create minimal API for workspace, flows, entries;
- create audit log base.

## Forbidden

- no legacy finance data import;
- no UI polish;
- no charts;
- no forecast;
- no assistant workflow;
- no legacy finance logic.

## Exit criteria

- workspace can be created;
- Cash/Card flows can be created;
- entries can be created and read;
- category seed loads;
- audit log base exists;
- QA confirms foundation.

## Handoff to Sprint 03

Include schema status, API status, backend risks, tests, and files changed.
