# Sprint 04 — Summary, Reports, and Month Control

## Goal

Build generated reports from entries and month-control behavior.

## Depends on

Sprint 03 final report.

## Director rule

Use a new Director. Start from project docs and Sprint 03 handoff only.

## Agents

- Financial Logic Engine Agent
- Data and Backend Core Agent
- QA/Audit Agent

## Scope

- monthly summary;
- category-by-month matrix;
- yearly summary base;
- Other expenses report;
- month close;
- correction / recalculate / cancel behavior;
- report API.

## Forbidden

- no manual numeric report editing;
- no UI dashboard expansion;
- no full archive import;
- no forecast.

## Exit criteria

- monthly summary is generated from entries;
- category matrix works;
- closed month cannot silently change;
- Other expenses report works;
- QA confirms formula consistency.

## Handoff to Sprint 05

Include report API status, month states, formula checks, known report risks, and files changed.
