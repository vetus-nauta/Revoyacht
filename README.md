# Revoyacht

Revoyacht is the working repository for RevoYacht / Yacht Flex Deck: a yacht-first operational digital twin and structure builder.

Current live demo:

https://brkovic.ltd/yacht-flex-demo/?v=20260507-v40d3d4g-builder-deck-badges

Current focus:

- Builder V2 for motor monohull / catamaran / trimaran structures
- ready-made motor yacht structure as the interaction baseline
- shared decks, hull-specific decks, areas/zones, equipment and instances
- future service/cardholder hooks without turning Yacht Flex Deck into a service app yet

Repository layout:

```text
private/yacht-flex-deck/          private deployment snapshot
public_html/yacht-flex-demo/      public demo deployment snapshot
docs/                             continuation briefs and product notes
```

Current version:

